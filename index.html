Battleship

Como Jogar

  1. Inicie o jogo conforme as instruções do README
  2. Entre com o nome do primeiro jogador
  3. Escolha entre as opções 1 para a colocação manual e 2 para a colocação aleatória dos barcos no tabuleiro
  4. Veja como dispor os barcos aqui
  5. Ao iniciar, escolha o número correspondente à linha que gostaria de atacar
  6. Depois, escolha o número correspondente à coluna que gostaria de atacar
  7. Caso acerte o alvo, continue atirando
  8. Caso erre o alvo, passe a vez para o coleguinha

Colocação Manual dos Barcos:

  • Escolha H para colocar o barco na Horizontal e V para colocar o barco na Vertical
  • Escolha o número correspondente a linha que gostaria de colocar a primeira posição do barco
  • Escolha o número correspondente a coluna que gostaria de colocar a primeira posição do barco
  • Faça o mesmo para todos os barcos
  • Ao final, escolha Y para concluir ou N para refazer a colocação dos barcos

Colocação aleatória dos Barcos:

  • Escolha Y caso aceite a disposição selecionada e N para gerar outra disposição
